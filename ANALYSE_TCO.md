# Analyse du Coût Total de Possession (TCO)

Cette analyse détaille la méthodologie utilisée pour l'optimisation financière de la flotte de véhicules.

## 1. Déterminer le Loyer Financier (La Base)

Le loyer est calculé selon trois variables principales :
- **La Valeur d'Achat** : Le prix négocié du véhicule (remise flotte incluse).
- **La Valeur Résiduelle (VR)** : C'est l'estimation du prix de revente du véhicule à la fin du contrat. Plus la VR est élevée, plus le loyer est bas.
- **La Durée et le Kilométrage** : Un contrat de 48 mois avec 150 000 km coûtera plus cher qu'un contrat de 36 mois avec 60 000 km à cause de la dépréciation.

### Formule simplifiée :
$$Loyer \approx \frac{(Prix\ d'achat - Valeur\ Résiduelle) + Intérêts}{Nombre\ de\ mois}$$

---

## 2. Intégrer les Services (Leasing Opérationnel)

Dans une flotte, on utilise souvent la **Location Longue Durée (LLD)** qui inclut des services pour lisser les coûts :
- **Maintenance et pneumatiques** : Forfait mensuel selon l'usage intensif.
- **Assurance flotte** : Souvent négociée globalement.
- **Véhicule de remplacement** : Crucial pour ne pas interrompre l'activité.

---

## 3. La Fiscalité : L'élément critique

C'est ici que l'optimisation se joue. Pour une entreprise, il faut calculer :
- **La TVA** : Récupérable sur le loyer et le carburant (selon le type de véhicule, utilitaire vs tourisme).
- **La TVS (Taxe sur les Véhicules de Sociétés)** : Désormais remplacée par deux taxes basées sur les émissions de $CO_2$ et l'ancienneté.
- **L'Amortissement Non Déductible (AND)** : Une part du loyer peut ne pas être déductible si le véhicule est trop polluant ou trop cher.

---

## 4. Le Calcul du TCO (Le vrai coût)

Pour comparer deux véhicules, ne regardez pas le loyer, regardez le **TCO (Total Cost of Ownership)**.

| Composante | Description |
| :--- | :--- |
| **Loyer LLD** | Financement + Services + Valeur de perte. |
| **Énergie** | Carburant ou électricité (poste majeur en transport). |
| **Fiscalité** | Taxes $CO_2$ + Charges sociales sur avantages en nature. |
| **Coûts indirects** | Frais de gestion de flotte, amendes, sinistralité. |
